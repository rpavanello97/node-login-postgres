# Login project with NODE and POSTGRES

#### Project was created to test a login functionality with node and postgres sql.

### Steps to start:

    - npm init -y    
    - npm i express
    - npm i -D nodemon
    - npm i ejs
    - used skeleton on html
    - npm i dotenv pg
    - npm i bcrypt
    - npm i express-session express-flash
    - npm i passport passport-local
